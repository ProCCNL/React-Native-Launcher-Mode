# react-native-launcher-mode

Launcher Mode for React Native

## Installation

```sh
npm install react-native-launcher-mode
```

## Usage

```js
import LauncherMode from "react-native-launcher-mode";

// ...

const result = await LauncherMode.multiply(3, 7);
```
